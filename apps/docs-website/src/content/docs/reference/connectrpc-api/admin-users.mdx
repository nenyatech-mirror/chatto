## AdminUserService

Server-admin user management commands and member-detail reads.

<a id="chatto-admin-v1-AdminUserService-ListMembers"></a>

### ListMembers

Lists server members for the admin members screen. Requires
admin.view-users.

```http
POST /api/connect/chatto.admin.v1.AdminUserService/ListMembers
```

#### Input: ListMembersRequest

Request server-admin member rows.

| Field | Type | Description |
| --- | --- | --- |

<a id="chatto-admin-v1-AdminUserService-UpdateUser"></a>

### UpdateUser

Updates another user's login and/or display name as an admin action.
Requires user.manage-accounts; the caller cannot target their own account.

```http
POST /api/connect/chatto.admin.v1.AdminUserService/UpdateUser
```

<a id="chatto-admin-v1-UpdateUserRequest"></a>

#### Input: UpdateUserRequest

Request to update a user's identity as a server-admin action.

| Field | Type | Description |
| --- | --- | --- |
| `user_id` | `string` | Target user ID. |
| `display_name` | `optional string` | New display name, when changing it. Empty clears the explicit display name. The server also rejects control and confusing invisible characters. |
| `login` | `optional string` | New login identifier, when changing it. The server accepts ASCII letters, digits, period, underscore, and hyphen, starting with a letter or digit. |


<a id="chatto-admin-v1-UpdateUserResponse"></a>

#### Result: UpdateUserResponse

Result of an admin user update.

| Field | Type | Description |
| --- | --- | --- |
| `user` | [`chatto.api.v1.User`](/reference/connectrpc-api/types/#chatto-api-v1-User) | Updated user profile. |
