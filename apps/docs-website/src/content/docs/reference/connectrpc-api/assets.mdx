## AssetService

Reads room-scoped asset metadata and freshly signed asset URLs.

<a id="chatto-api-v1-AssetService-GetAsset"></a>

### GetAsset

Reads one asset. Authentication and room membership are required. Returns
NOT_FOUND when the asset is missing, deleted, or owned by a different room.

```http
POST /api/connect/chatto.api.v1.AssetService/GetAsset
```

<a id="chatto-api-v1-GetAssetRequest"></a>

#### Input: GetAssetRequest

Request to read one room-scoped asset with freshly signed URLs.

| Field | Type | Description |
| --- | --- | --- |
| `room_id` | `string` | Required. Room that owns the asset. |
| `asset_id` | `string` | Required. Asset ID. |
| `thumbnail` | [`ImageTransformOptions`](/reference/connectrpc-api/types/#chatto-api-v1-ImageTransformOptions) | Thumbnail URL options. Defaults are applied when absent. |


<a id="chatto-api-v1-GetAssetResponse"></a>

#### Result: GetAssetResponse

Response containing one room-scoped asset.

| Field | Type | Description |
| --- | --- | --- |
| `asset` | [`Asset`](/reference/connectrpc-api/types/#chatto-api-v1-Asset) | Asset metadata and signed URLs. |


<a id="chatto-api-v1-AssetService-BatchGetAssets"></a>

### BatchGetAssets

Reads many assets in one room. Authentication and room membership are
required. Missing, deleted, and wrong-room asset IDs are omitted. Results
preserve first-seen request order and repeated asset IDs are de-duplicated.

```http
POST /api/connect/chatto.api.v1.AssetService/BatchGetAssets
```

<a id="chatto-api-v1-BatchGetAssetsRequest"></a>

#### Input: BatchGetAssetsRequest

Request to read many room-scoped assets with freshly signed URLs.

| Field | Type | Description |
| --- | --- | --- |
| `room_id` | `string` | Required. Room that owns the assets. |
| `asset_ids` | `repeated string` | Required. Asset IDs. Missing, deleted, and wrong-room asset IDs are omitted. |
| `thumbnail` | [`ImageTransformOptions`](/reference/connectrpc-api/types/#chatto-api-v1-ImageTransformOptions) | Thumbnail URL options. Defaults are applied when absent. |


<a id="chatto-api-v1-BatchGetAssetsResponse"></a>

#### Result: BatchGetAssetsResponse

Response containing room-scoped assets.

| Field | Type | Description |
| --- | --- | --- |
| `assets` | repeated [`Asset`](/reference/connectrpc-api/types/#chatto-api-v1-Asset) | Assets in first-seen request order. |
