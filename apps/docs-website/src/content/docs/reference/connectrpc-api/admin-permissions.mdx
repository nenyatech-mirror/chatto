## AdminPermissionService

Provides permission matrix reads and admin permission writes.

<a id="chatto-admin-v1-AdminPermissionService-GetRolePermissionTierMatrix"></a>

### GetRolePermissionTierMatrix

Gets the role-permission matrix for a single tier. Requires role.manage
for server/group scope; room scope requires role.manage or room.manage in
that room.

```http
POST /api/connect/chatto.admin.v1.AdminPermissionService/GetRolePermissionTierMatrix
```
