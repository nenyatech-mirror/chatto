## ServerMemberService

Reads authenticated server members.

<a id="chatto-api-v1-ServerMemberService-ListMembers"></a>

### ListMembers

Lists authenticated server members. Every authenticated user is a server
member; admin-sensitive fields stay out of this public row shape.

```http
POST /api/connect/chatto.api.v1.ServerMemberService/ListMembers
```

<a id="chatto-api-v1-ListServerMembersRequest"></a>

#### Input: ListServerMembersRequest

Request for server members visible to the authenticated user.

| Field | Type | Description |
| --- | --- | --- |
| `search` | `string` | Optional case-insensitive search against login and display name. |
| `page` | [`PageRequest`](/reference/connectrpc-api/types/#chatto-api-v1-PageRequest) | Page request. Defaults to 20 results when absent or limit is zero. |


<a id="chatto-api-v1-ListServerMembersResponse"></a>

#### Result: ListServerMembersResponse

Server member page.

| Field | Type | Description |
| --- | --- | --- |
| `members` | repeated [`DirectoryMember`](/reference/connectrpc-api/types/#chatto-api-v1-DirectoryMember) | Members in the requested page. |
| `page` | [`PageInfo`](/reference/connectrpc-api/types/#chatto-api-v1-PageInfo) | Page metadata. |
