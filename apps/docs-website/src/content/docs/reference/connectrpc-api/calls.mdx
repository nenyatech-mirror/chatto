## VoiceCallService

Voice-call request/response APIs backed by Chatto call-state projections.

<a id="chatto-api-v1-VoiceCallService-ListActiveCallRooms"></a>

### ListActiveCallRooms

Lists channel room IDs that currently have active calls.

Returns an empty list when LiveKit is not configured.

```http
POST /api/connect/chatto.api.v1.VoiceCallService/ListActiveCallRooms
```

<a id="chatto-api-v1-ListActiveCallRoomsRequest"></a>

#### Input: ListActiveCallRoomsRequest

Request for active channel call room IDs.

Active channel call room IDs.

| Field | Type | Description |
| --- | --- | --- |
| `room_ids` | `repeated string` | Room IDs with active calls. |

### ListCallParticipants

Lists participants currently observed in a room's active call.

POST /api/connect/chatto.api.v1.VoiceCallService/ListCallParticipants

<a id="chatto-api-v1-VoiceCallService-GetCallToken"></a>

### GetCallToken

Issues a LiveKit token for joining the room's active call.

The caller must be a member of the room and a call must already be active.

```http
POST /api/connect/chatto.api.v1.VoiceCallService/GetCallToken
```

<a id="chatto-api-v1-GetCallTokenRequest"></a>

#### Input: GetCallTokenRequest

Request for a LiveKit token for a room call.

| Field | Type | Description |
| --- | --- | --- |
| `room_id` | `string` | Required. Room whose active call should be joined. |


<a id="chatto-api-v1-GetCallTokenResponse"></a>

#### Result: GetCallTokenResponse

LiveKit token details for joining a room call.

| Field | Type | Description |
| --- | --- | --- |
| `token` | `string` | LiveKit JWT token. |
| `e2ee_key` | `string` | Shared E2EE key for this active call. |
| `call_id` | `string` | Active call session ID. |
