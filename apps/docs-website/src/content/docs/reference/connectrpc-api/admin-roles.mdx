## AdminRoleService

Provides authenticated role catalog reads and admin role CRUD.

### ListRoles

Lists the server role catalog. Requires an authenticated user.

```http
POST /api/connect/chatto.admin.v1.AdminRoleService/ListRoles
```

<a id="chatto-admin-v1-ListRolesRequest"></a>

#### Input: ListRolesRequest

Request the server role catalog.


<a id="chatto-admin-v1-ListRolesResponse"></a>

#### Result: ListRolesResponse

Server role catalog plus viewer role-management capabilities.

| Field | Type | Description |
| --- | --- | --- |
| `roles` | repeated [`Role`](/reference/connectrpc-api/types/#chatto-admin-v1-Role) | Roles sorted by position. |
| `viewer_can_manage_roles` | `bool` | Whether the caller may create/update/delete role definitions. |
| `viewer_can_assign_roles` | `bool` | Whether the caller may assign/revoke roles and view role rosters. |
