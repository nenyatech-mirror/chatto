## RoomMemberService

Reads explicit room members.

<a id="chatto-api-v1-RoomMemberService-ListMembers"></a>

### ListMembers

Lists explicit members of a room. The caller must be a member of the room.

```http
POST /api/connect/chatto.api.v1.RoomMemberService/ListMembers
```

<a id="chatto-api-v1-ListRoomMembersRequest"></a>

#### Input: ListRoomMembersRequest

Request for members of one room.

| Field | Type | Description |
| --- | --- | --- |
| `room_id` | `string` | Required. Room whose explicit members should be listed. |
| `search` | `string` | Optional case-insensitive search against login and display name. |
| `page` | [`PageRequest`](/reference/connectrpc-api/types/#chatto-api-v1-PageRequest) | Page request. Defaults to 20 results when absent or limit is zero. |

### GetMember

Gets one explicit member of a room. The caller must be a member of the

```http
POST /api/connect/chatto.api.v1.RoomMemberService/GetMember
```
