## ExternalIdentityAuthService

Public external identity authentication operations.

<a id="chatto-auth-v1-ExternalIdentityAuthService-GetPendingExternalIdentity"></a>

### GetPendingExternalIdentity

Reads safe metadata for a pending create/link flow. This public RPC
requires a valid pending-flow capability token and returns NOT_FOUND when
the token is unknown or expired.

```http
POST /api/connect/chatto.auth.v1.ExternalIdentityAuthService/GetPendingExternalIdentity
```

<a id="chatto-auth-v1-GetPendingExternalIdentityRequest"></a>

#### Input: GetPendingExternalIdentityRequest

Request to inspect a pending external identity flow.

| Field | Type | Description |
| --- | --- | --- |
| `token` | `string` | Raw pending flow token from the provider callback redirect. |


<a id="chatto-auth-v1-GetPendingExternalIdentityResponse"></a>

#### Result: GetPendingExternalIdentityResponse

Result of inspecting a pending external identity flow.

| Field | Type | Description |
| --- | --- | --- |
| `pending` | [`PendingExternalIdentity`](#chatto-auth-v1-PendingExternalIdentity) | Pending flow metadata. |


<a id="chatto-auth-v1-ExternalIdentityAuthService-CreateExternalIdentityAccount"></a>

### CreateExternalIdentityAccount

Creates a passwordless account from a pending provider identity. This
public RPC requires a valid create-flow capability token.

```http
POST /api/connect/chatto.auth.v1.ExternalIdentityAuthService/CreateExternalIdentityAccount
```

| Field | Type | Description |
| --- | --- | --- |

| `login` | `string` | Desired username. |

## Supporting Types

<a id="chatto-auth-v1-PendingExternalIdentity"></a>

### PendingExternalIdentity

Metadata about a pending external identity flow. The provider subject is not
exposed; clients only need user-facing provider/profile hints.

| Field | Type | Description |
| --- | --- | --- |
| `kind` | [`ExternalIdentityFlowKind`](#chatto-auth-v1-ExternalIdentityFlowKind) | Pending flow kind. |
| `provider_id` | `string` | Provider ID that produced the identity. |
| `provider_type` | `string` | Provider type that produced the identity. |
| `provider_label` | `string` | Provider label shown to the user. |
| `verified_email` | `string` | Verified email supplied by the provider, when available. |
| `login_hint` | `string` | Suggested username derived from provider profile hints. |
| `display_name_hint` | `string` | Suggested display name derived from provider profile hints. |
| `bound_user_id` | `string` | User ID this link flow is bound to. Empty for create flows. |
| `redirect_path` | `string` | Internal path the client should return to after confirmation. |




## Enums

<a id="chatto-auth-v1-ExternalIdentityFlowKind"></a>

### ExternalIdentityFlowKind

Mode of a pending external identity confirmation flow.

| Name | Number | Description |
| --- | --- | --- |
| `EXTERNAL_IDENTITY_FLOW_KIND_UNSPECIFIED` | `0` | The flow kind was not specified. |
| `EXTERNAL_IDENTITY_FLOW_KIND_CREATE_ACCOUNT` | `1` | Create a new passwordless account and link the provider identity. |
| `EXTERNAL_IDENTITY_FLOW_KIND_LINK_ACCOUNT` | `2` | Link the provider identity to the authenticated account. |
